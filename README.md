# Aurora Ridge Partners - Personal Website

Personal website for Darian Vale Koh, Founder & Chairman of Aurora Ridge Partners.

## Features
- **Premium Design**: Minimalist and institutional-grade aesthetic.
- **Responsive Layout**: Designed for all screen sizes (mobile and desktop).
- **High-Quality Visuals**: Custom-selected architectural imagery and professional portraits.
- **Institutional Branding**: Custom SVG favicon and typography (Cormorant Garamond & Inter).

## Tech Stack
- HTML5
- Vanilla CSS
- Modern SVG Icons

## Deployment
This site is designed to be hosted on **GitHub Pages**. All image references are relative to ensure seamless loading across different hosting environments.
